# BilaTraining.Web

React frontend for BilaTraining, built with Vite.

## Scripts

```bash
npm start
npm run build
```

## Local development

By default the app expects the API at:

```text
http://localhost:5175/api
```

Override it with a local env file:

```env
VITE_API_BASE_URL=http://localhost:5175/api
```
